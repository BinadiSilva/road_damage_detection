# Road Damage Detection System

## Module
ICT2403 – Graphics and Image Processing  
Department of Computing  
Rajarata University of Sri Lanka  

## Project Title
Video-Based Road Surface Damage Detection and Severity Analysis

## Objective
The objective of this project is to develop a video-based image processing system to detect, segment, and analyze road surface damages such as potholes, cracks, and surface deterioration using computational techniques.

## Project Phases
1. Video Acquisition
2. Frame Enhancement
3. Segmentation
4. Evaluation

## Technologies Used
- Python
- OpenCV
- NumPy
- Matplotlib

## Git Workflow
- main: Final stable version
- dev: Development branch
- feature branches for each phase

All merges to the main branch require approval from the project leader.
